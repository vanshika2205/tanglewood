## 🛠️ Design & Code Quality Standards

To make these projects feel **extremely premium** and **industry-grade**, we will strictly adhere to the following rules:
1. **Tailored HSL Colors:** No default raw primary colors (like `#FF0000` or `#0000FF`). We will use custom premium palettes (e.g., deep dark blues, glowing neon accents, warm glassmorphism overlays).
2. **Typography Excellence:** We will import modern typography (Inter, Outfit, Fira Code, Playfair Display) from Google Fonts.
3. **Fluid Micro-Animations:** Every button hover, state change, and modal open will have bespoke CSS transitions and smooth ease-in-out properties.
4. **Clean, Componentized Code:** Highly structured JS using ES6 modules (where appropriate), and clean CSS structure with custom variables (`:root`).
5. **Zero Placeholders:** Real, high-fidelity mock data and generated visual assets so every app looks 100% complete and production-ready.
